# GRC Risk Assessment — NovaPay Fintech Ltd.

A formal Information Security Risk Assessment for a fictional fintech startup, built as a portfolio project to demonstrate GRC analyst skills.

## What's Inside

- 12 identified risks with likelihood, impact, and risk scores
- Color-coded risk register (Critical / High / Medium / Low)
- Controls mapped to ISO 27001:2022 Annex A
- GDPR compliance obligations (Articles 5, 25, 28, 32, 33, 35)
- Statement of Applicability (SoA)
- Full Risk Treatment Plan with owners and target dates

## Frameworks Applied

- ISO 27001:2022
- GDPR
- NIST CSF
